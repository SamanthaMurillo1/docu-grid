# DocuGrid

This contains everything you need to run DocuGrid locally.

View your app in AI Studio: https://ai.studio/apps/174df047-6480-4de5-86b9-fd61580406e2

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set `GEMINI_API_KEY` and `APP_URL` in [.env](.env)
3. Run the app:
   `npm run dev`
